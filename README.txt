LWROpenIGTIF is a open source Interface package for the LWR Iiwa KUKA laboratories GmbH which is allows to communicate via OpenIGTLink.
There are to Interfaces:
	- Visualisation Interface: Via this Interface the current robot pose and the estimated force at the TCP can be send
	- State machine Interface: Via this interface a bidirectional communication with a state control can be established to operate a state machine on the robot control.
